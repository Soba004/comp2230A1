Instructions

Compile each program
- javac MazeGenerator.java
- javac MazeSolver.java
- javac MazeVerifier.java

Running each program

1) MazeGenerator

- java MazeGenerator numberOfRows numberOfColumns outputFileName

where 
numberOfRows is a positive integer
numberOfColumns is a positive integer
outputFileName is a file name

2) MazeSolver

- java MazeSolver inputFileName outputFileName

where
inputFileName is a file name

BFS solution will always be solved/displayed first
DFS solution will always be solved/displayed second

3) MazeVerifier
- java MazeVerifier inputFilename1 inputFileName2
